# 小節

Cocos Creator 是一個基於 Cocos 引擎的開發套件

在應用上比單純對Cocos 引擎敲代碼方便

佈局含邏輯上很像Unity引擎，所以由 Unity 跨到 Cocos Creator 是很方便

加上Cocos Creator使用的是 JavaScript 或是 TypeScript

在撰寫上比C++ C#方便一些，

但Cocos Creator還是難以跟Unity相比

但在開發純2D上Cocos Creator不失為一選擇
